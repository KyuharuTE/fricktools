# ✅ 实现功能
- ADB类
  - 重启至FastBoot
  - 重启至Recovery
  - 重启
  - 检测设备连接
  - 安装APK
  - Shell终端
  - 截图
  - 无线调试
  - 激活APP菜单(ADB)
    - Shizuku
    - 冰箱（Ice Box）
- FastBoot类
  - 检测设备连接 
  - 重启至系统
  - 重启至Recovery
- 一键刷入类
  - 刷入Recovery
  - 刷入Boot
  - 一键刷入线刷包
- 小工具
  - 打开一个具备ADB环境的终端窗口
  - 解压线刷包
